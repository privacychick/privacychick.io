# Perplexity AI Privacy Research Notes

## What Perplexity Collects
- Device data and interaction data from all users
- Personal info stored if you create an account
- Search queries used for AI training (opt-out available)
- IP address, device fingerprint

## Privacy Settings Available
1. **AI Data Usage toggle** — Profile > Settings > "AI Data Usage" — turn OFF to stop queries being used for AI training
2. **Incognito Mode** — threads expire after 24 hours, not saved to history. Access via profile icon > Incognito toggle
3. **No account usage** — using without an account limits personal data tied to a profile

## Critical Caveat — April 2026 Lawsuit
- Federal lawsuit filed April 2026: *Doe v. Perplexity AI Inc., 3:26-cv-02803, N.D. California*
- Alleges Perplexity embedded Facebook Meta Pixel, Google Ads, Google DoubleClick, and Meta's Conversions API
- Claims full conversation text transmitted to Meta and Google — including in Incognito Mode
- Lawsuit calls Incognito Mode "a sham" — same trackers allegedly fired regardless
- Perplexity has not denied the tracking practices, only said they hadn't been served the lawsuit
- Pattern of behavior: robots.txt violations (2024), stealth crawlers (2025), now this

## Honest Assessment for Privacy Chick Page
Perplexity is a genuinely useful and capable AI search tool, but it has serious, documented privacy concerns:
- Incognito mode may NOT protect you from third-party tracking (pending lawsuit)
- Queries used for AI training by default (can opt out)
- Third-party advertising trackers potentially present

## Recommended Approach for the Page
Be honest: Perplexity is powerful but not private. Recommend it with clear caveats:
1. Turn off AI Data Usage in Settings
2. Use Incognito Mode for sensitive searches (with caveat about its limitations)
3. Never enter truly sensitive info (health, financial, legal) into any cloud AI
4. Consider using without creating an account

## Alternative framing

- Primary recommendation: Perplexity (with privacy settings guide)
